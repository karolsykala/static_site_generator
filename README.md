# Static Site Generator in Python

A simple static site generator built with Python. This project converts Markdown files into HTML pages using a customizable template system.
## Features

  - Convert Markdown files (.md) to HTML.
  - Use templates for consistent page layout.
  - Lightweight and easy to extend.

## Installation

    Clone the repository:

git clone https://github.com/karolsykala/static-site-generator-python.git

Install dependencies:

    pip install -r requirements.txt

Usage

  1. Place your Markdown files in the content directory.
  2. Create a template in the templates directory.
  3. Run the generator:

    python generate.py

  4. View the output in the output directory.
